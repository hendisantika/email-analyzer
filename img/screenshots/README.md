## Guidelines for Adding Screenshots

When adding new screenshots:
1. Use PNG format for best quality
2. Capture at 1920x1080 or similar resolution
3. Use descriptive filenames (e.g., `history-page.png`, `dashboard.png`)
4. Update the main README.md to reference new screenshots
5. Optimize images if they exceed 2MB using tools like TinyPNG
